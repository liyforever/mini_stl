MiniSTL
====
采用C++11实现的简易STL标准库，既是C++STL的一个子集(没有实现标准IO库等)又是一个超集(增加了一些容器和算法)

目的：练习数据结构与算法和C++ Template编程

编译环境:VS2012

开发计划:
=========
* STL的几大基本组建,vector、list、deque、set、map、unordered_*等
* STL算法库中的所有算法
* AVL_set、AVL_multiset、AVL_map、AVL_multimap
* shared_ptr
